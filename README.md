# URL界的算命仙
製作人:周聖詠、許翊泓、姚承希

## 使用步驟:
1. Download zip
2. 到你的電腦解壓縮，開啟虛擬機至本資料夾
3. 輸入指令
```
pip install -r requirements.txt
```
4. 啟動程式
```
python run.py
```
